# Java To-Do List

A simple **Java Swing To-Do List application** that allows users to add, view, and remove tasks through an interactive GUI.

## Features
- Add new tasks
- View tasks in a scrollable list
- Remove selected tasks
- Simple and clean user interface
- Built using Java Swing

## Technologies Used
- Java
- Swing (GUI)

## Project Structure
- `todo_list.java` → Main source code file
